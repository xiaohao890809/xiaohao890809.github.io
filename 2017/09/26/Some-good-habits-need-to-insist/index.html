保持某些好的习惯

发表于 2017-09-26 | 更新于 2017-12-19 | 分类于 生活

优秀是一种习惯，保持一个良好的习惯往往能引导一种健康的生活姿态，现在记录一些自己认为是比较好的一些习惯，自己经常去看看，用于监督和督促自己。

生活

  1. 一周至少运动两次（跑步，爬山，打球等）
  2. 晚上11点半准备看书，然后睡觉
  3. 一天至少三大杯水
  4. 平时多吃点水果
  5. 一个月清理一次房间
  6. 晚上睡前刷牙

技术

  1. 代码写完后多检查下注释有没有写全，没写的补上，以免以后都不知道自己当初写的啥
  2. LeetCode过三遍
  3. js和go语言了解下
  4. 正则表达式多熟悉

反思

  1. 每天问一遍自己，想进BAT吗，以你现在的能力能进BAT吗，不能的话哪些地方需要加强呢？

工作

  1. 平时多看下面试相关的信息，刷刷题
  2. 把跟自己业务相关的数据库表多检查下

英语

  1. 熟读新概念英语第三册前30篇
记录博客的第一天
